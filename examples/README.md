## How to Use These Examples

### 1. See What to Expect
Review examples before using a skill to understand the output format and level of detail.

### 2. Compare Your Results
Use examples as a quality benchmark for your own skill outputs.

### 3. Customize for Your Needs
See how examples are structured, then adapt templates for your team.

## Example Format

Each example includes:
- **Input summary** - What was being analyzed
- **Full output** - Complete skill analysis
- **Notes** - Key takeaways or customization tips

## Testing Your Skills

Want to test skills before using in production?

1. Pick a relevant example from this directory
2. Use it as input for the skill
3. Compare output quality to the example
4. Iterate on your prompt if needed

## Example Quality Levels

Examples demonstrate three quality levels:

**⭐ Basic** - Meets minimum requirements
- Follows template structure
- Provides actionable feedback
- Includes prioritization

**⭐⭐ Good** - Production quality
- Comprehensive analysis
- Specific, detailed recommendations
- Evidence-backed insights
- Educational tone

**⭐⭐⭐ Excellent** - Best-in-class
- Deep insights beyond template
- Context-aware recommendations
- Anticipates edge cases
- Clearly actionable with owners

All examples in this directory aim for "Good" or better.
